# 06-部署运行文档

## 1. 环境要求

| 组件 | 版本要求 | 说明 |
|---|---|---|
| JDK | 17+ | 编译和运行 |
| Maven | 3.6+ | 构建工具 |
| MySQL | 8.0+ | 关系数据库，端口 3306 |
| Redis | 6/7 | 缓存和滑动窗口，端口 6379 |
| RabbitMQ | 3.x | 消息队列，端口 5672 / 15672 |

---

## 2. 基础服务启动

### 2.1 MySQL

确保 MySQL 服务已启动，端口 3306 可访问。

### 2.2 Redis

```bash
# Linux / macOS
redis-server

# Windows
redis-server.exe
```

验证：

```bash
redis-cli PING
# 返回 PONG 表示正常
```

### 2.3 RabbitMQ（Docker 推荐）

```bash
docker run -d --name rabbitmq \
  -p 5672:5672 \
  -p 15672:15672 \
  -e RABBITMQ_DEFAULT_USER=guest \
  -e RABBITMQ_DEFAULT_PASS=guest \
  rabbitmq:3.13-management
```

- 管理后台：http://localhost:15672
- 账号：`guest` / `guest`

## 4. 配置文件说明

### 4.1 application.yml

```yaml
server:
  port: 8080

spring:
  datasource:
    url: jdbc:mysql://localhost:3306/charge_monitor?...
    username: ${MYSQL_USERNAME:}       # 环境变量，空则默认空
    password: ${MYSQL_PASSWORD:}

  data:
    redis:
      host: localhost
      port: 6379
      password: ${REDIS_PASSWORD:}      # 环境变量

  rabbitmq:
    host: localhost
    port: 5672
    username: guest
    password: guest
```

### 4.2 环境变量

```bash
# Linux / macOS
export MYSQL_USERNAME=root
export MYSQL_PASSWORD=your_password
export REDIS_PASSWORD=                # 无密码可留空

# Windows（PowerShell）
$env:MYSQL_USERNAME="root"
$env:MYSQL_PASSWORD="your_password"
```

或者在 IDEA 的 Run Configuration → Environment Variables 中配置。

## 5. 启动项目

### 5.1 IDEA 启动（推荐）

打开 `ChargeMonitorApplication.java`，右键 → Run。

### 5.2 命令行启动

```bash
cd charge-monitor-backend

# 编译
mvn clean package -DskipTests

# 启动
mvn spring-boot:run
```

## 8. 常见问题

### 8.1 MySQL 连接失败

```
Access denied for user 'root'@'localhost'
```

检查 `MYSQL_USERNAME` / `MYSQL_PASSWORD` 环境变量是否正确。

### 8.2 RabbitMQ 连接失败

```
java.net.ConnectException: Connection refused: getsockopt
```

确认 RabbitMQ 已启动，端口 5672 可访问。

### 8.3 WebSocket 连接被 CSP 拦截

浏览器控制台出现 Content Security Policy 错误。

解决方法：在 `http://localhost:8080/doc.html` 页面打开控制台连接，而不是在 MSN/Bing 等外部页面。

### 8.4 Redis 连接失败

确认 Redis 服务已启动，端口 6379 可访问。

### 8.5 编译报错 Lombok

```
java.lang.ExceptionInInitializerError: com.sun.tools.javac.code.TypeTag :: UNKNOWN
```

检查 JDK 版本是否为 17。JDK 21+ 需要升级 Lombok 版本。
